- 여기서 리눅스 Shell을 Open.
  혹은  프로젝트 위치로 이동 
  cd /mnt/d/GitHub/CCd.Meshoptimizer
  
- 아래 명령 수행.  
cmake -B build_linux -DCMAKE_BUILD_TYPE=Release -DMESHOPT_BUILD_CCd_LINUX=ON -DMESHOPT_INSTALL=OFF
cmake --build build_linux --parallel


[TEST]
gcc test/test_so.c \
    -L./build_linux \
    -lCCdMeshOptimizer \
    -Wl,-rpath,./build_linux \
    -o test/test_so

// 이렇게 실행하면, 압축된 glb가 나와야 성공.
./test/test_so -i ./test/data/tree.glb -o ./test/data/treeout.glb -c -tc



[Docker 만들기]
컴파일 성공하면
docker build -t ccd-meshoptimizer .
